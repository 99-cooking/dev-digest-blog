## ðŸ”® EMERGING TRENDS

**Skills > MCP for Agent Knowledge**
Nx's blog post crystallizes something many are noticing: giving agents structured knowledge files ("skills") is often more effective than spinning up MCP servers. MCP is great for tool connectivity, but overfitting every interaction to tool calls adds latency and complexity. This "skills-first" approach aligns with how Claude Code's own CLAUDE.md works.

**Sonnet-Class Models Eating Opus's Lunch**
Claude Sonnet 4.6 approaching Opus-level performance at Sonnet pricing is a pattern we've seen accelerating. The capability gap between model tiers is shrinking fast. For most coding tasks, the mid-tier model is becoming "good enough" â€” save Opus for the truly hard stuff.

**AI Coworkers with Memory**
Rowboat (6.2K stars, trending) and similar projects show growing demand for AI tools that remember context across sessions. Local-first, knowledge-graph-based approaches are winning over cloud-dependent alternatives. The fact that Rowboat runs entirely locally with Markdown files is very much the zeitgeist.
